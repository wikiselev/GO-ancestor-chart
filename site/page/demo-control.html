• Introduction

    Welcome to QuickGO, I'm David Binns, a member of the QuickGO team.
    GO slims are a collection of high level GO terms which can be used to categorise gene products.
    Please type nucleus into the main search box

  • Select 'Nucleus' term

    To make a custom slim, a set of GO terms are first selected
    Press the button next to GO:005634 nucleus

  • This shows your currently selected terms

    Press close

  • Find 'Extracellular space' term

    Please type extracellular into the main search box

  • Select 'Extracellular space' term

    Press the button next to GO:005615 extracellular space

  • Use these terms

    Press the 'Use terms' button to use compare terms in your selection and find annotations based on these terms.

  • View information about 'extra cellular space'

    Click on GO:0005615 to view information about the term 'extracellular space'

  • Compare 'nucleus'

    Click on GO:0005634 to compare the term 'nucleus' to the previous term

  • Find annotations

    Click on the 'Find annotation' tab to view annotations made to gene products corresponding to the chosen terms.

  • Download annotation

    Click on the 'Download' icon to obtain a file containing these annotations

  • Start download

    Here you can choose what format the file should be in, how many lines it may contain, and if you want it gzipped.
    Click on the 'Download' button to start the file download.

  • Completed

    Your file should appear in your browser or download directory shortly.
    Well done, you have completed this tutorial.
